Smart Caching Layer API is a FastAPI-based demo that integrates Redis caching, SQLite storage, and AI-driven pre-caching. It demonstrates how to efficiently serve frequently accessed products and users using a cache-aside pattern, track cache hits/misses, manage TTLs, and automatically pre-load “hot” items predicted from access logs. The project showcases a lightweight approach to improving API performance with minimal infrastructure, including optional machine learning for smart caching.
